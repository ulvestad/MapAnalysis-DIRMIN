Main Page

The main page is where the process of preparing and scanning images is done. First the user has to select one or more folders containing .jpg-images and associated .xml-files, and prepare these files using the "Pre-process folders"-button. When this process is done, the user can scan all pre-processed data using the "Scan pre-processed data"-button, and see the results in the Result Page. The scanning will take some time, depending on the amount of images scanned at once.

The "Update known locations"-button lets the user select a .CSV file with coordinates to known quarries to update the database, in order to display them on the result page. After a scan i done, it is possible to get the results as a .CSV-file with the "Export new results"-button, and if the user wants to remove all pre-processed and scanned data from the system, the "Reset data"-button will do just that. This will also remove all information about quarries the user has confirmed using the Result Page.
